The Rules of the game.

The first five digits can be an unique number between 1 and 70.

The last number is a digit between 1 and 25 and does not have to be unique to the previous five numbers.

If I were wise I'd see a trend.

Currently the odds for winning this lottery are 302,575,350 to 1. Creating a JS function to create an array populated with Math.random() numbers with the rules above should lead to roughly those same odds.

These charts and graphs are meant to visualize the past winning sets to see if there's any sort of correlation in them.

The purpose isn't to write a function that merely gives you an array exactly equal to one of the winning sets, but to write a more general function that takes significantly fewer than 300,000,000 tries to match a given winning set. Ideally, any function that can consistently match a winning set within a few hundred tries is hopefully also spitting out future winning sets and possibly evening the odds of winning.
